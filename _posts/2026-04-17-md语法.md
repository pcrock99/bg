## {% raw %}用法
{% raw %}

这里可以随便写 {% 和 {{ 这些符号
它们不会被 Jekyll 解析

```html
<div>{{ variable }}</div>
{% endraw %}

> 这是一条提示
{: .prompt-tip }

> 这是一条普通说明
{: .prompt-info } 或 `{: .prompt-note }`

> 这是一个警告
{: .prompt-warning }

> 数据将永久删除
{: .prompt-danger }

## md折叠
```
<details>
  <summary>点击这里展开/收缩</summary>
  这里是被隐藏的内容。
  可以写多行文字，甚至包括 Markdown 格式。
</details>
```
