## {% raw %}用法
{% raw %}

这里可以随便写 {% 和 {{ 这些符号
它们不会被 Jekyll 解析

```html
<div>{{ variable }}</div>
{% endraw %}

## md折叠
```
<details>
  <summary>点击这里展开/收缩</summary>
  这里是被隐藏的内容。
  可以写多行文字，甚至包括 Markdown 格式。
</details>
```
