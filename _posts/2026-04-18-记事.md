# 检测歌曲链接
1. 打开 Chrome 浏览器
2. 按 F12 打开开发者工具，点 Console 标签
3. 复制下面的代码，粘贴进去，按回车：

```
// 把下面的链接替换成你的 playlist.txt 里的链接
let urls = [
    "https://example.com/song1.mp3",
    "https://example.com/song2.mp3"
];

urls.forEach(async (url) => {
    try {
        let response = await fetch(url, { method: "HEAD" });
        if (response.ok) {
            console.log("✅ 有效:", url);
        } else {
            console.log("❌ 失效 (" + response.status + "):", url);
        }
    } catch(e) {
        console.log("❌ 无法访问:", url);
    }
});
```
